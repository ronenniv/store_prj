## Implementation

Linux:
1. source venv/Scripts/activate
2. cd store_prj
3. pip install -r requirements.txt
4. define DATABASE_URL
5. define DEBUG_LEVE. Default is DEBUG
6. flask run

Windows:
1. venv/Scripts/activate.bat
2. cd store_prj
3. pip install -r requirements.txt
4. define DATABASE_URL
5. define DEBUG_LEVE. Default is DEBUG
6. flask run
